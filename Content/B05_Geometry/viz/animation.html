Dynamic Affine Interpolation

Point P, defined as A + t · (C − A), sweeps from A to C as t sweeps from 0 to 1. P is a point-vector sum, where the vector added to A starts at 0 when t is zero and increases to C − A when t reaches 1. Segment BP sweeps across the triangle. Lean computes every frame using exact rational arithmetic — the browser only plays them back.
