Architecture: This game implements the HtDP BigBang coinductive pattern — the same toDraw / onTick / onKey / stopWhen structure used in the Lean 4 version. The game loop is a coinductive process: each tick produces the next world state, running until stopWhen fires. All handler functions are pure.
